# Sentinel

Торговый бот для Binance (BTC/USDT, ETH/USDT) с машинным обучением
в качестве фильтра сигналов и жёстким риск-менеджментом.

## Что это, если коротко

Бот каждый час смотрит на свечи, считает индикаторы (RSI, MACD, ADX,
Bollinger и так далее), и если несколько стратегий одновременно
говорят «пора входить» — ML-модель решает, пропустить этот сигнал
или заблокировать как слабый. Дальше срабатывают риск-проверки,
и только после них бот реально открывает позицию.

По умолчанию бот работает в **paper-режиме** — то есть имитирует
сделки, но **не тратит ваши деньги**. Переключение на реальные
деньги — отдельное осознанное действие, не «галочка в настройках».

## Важно

Бот **не гарантирует прибыль**. Он снижает шансы сделать грубую
ошибку (переторговать, войти в явный шум, забыть поставить стоп),
но сам по себе не «печатный станок». Начинайте с paper-режима,
наблюдайте минимум две недели, и только потом думайте о реальных
деньгах.
